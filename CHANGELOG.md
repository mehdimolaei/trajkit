# Changelog

## [0.1.0] - 2025-12-28
### Added
- Initial public release of `trajkit`
- Core `Trajectory` / `TrajectorySet` data model with Parquet I/O helpers
- MSD utilities for single trajectories and trajectory sets
- Pair-weighted MSD aggregation, interpolation support, and per-dimension/mean-displacement outputs
- Conditional displacement/velocity (CDV) correlation pipeline for flow-field inference
- Example notebooks for CDV and trajectory workflows
- Developer tooling: tests via `pytest`, formatting/lint configs (black/ruff), docs scaffolding (MkDocs)
